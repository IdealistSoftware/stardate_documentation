Thanks for trying out Stardate!

To install the alpha version of Stardate, you'll need to install TestFlight on your iPhone, then accept the invitation and install the app.

### Install TestFlight

TestFlight is how Apple distributes apps to users before they're released to the public. It also allows you to send feedback to the developers if you find an issue or experience a crash.

import { LinkButton } from "@astrojs/starlight/components";

<LinkButton href="https://apps.apple.com/us/app/testflight/id899247664?mt=8">
  Download TestFlight from the App Store
</LinkButton>

### Accept Invitation

Once TestFlight is installed, you should be able to accept the invitation here and install Stardate.

<LinkButton href="https://testflight.apple.com/join/J4Ssh33V">
  Accept Invitation and Download Stardate
</LinkButton>

### Install and Configure Stardate

Once you've accepted the invitation, TestFlight will automatically install Stardate. Then you can continue on to confirm that it's installed on your Apple Watch and set up the watch complication.
